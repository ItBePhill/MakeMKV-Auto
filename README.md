# Make MKV-Auto
---------------------------------------------------------------
## Automatically finds and then rips blurays and dvds based on options in an ini file


> [!WARNING]
> In theory <b>should</b> work on linux however I am unable to try it

> [!WARNING]
> ### Requirements:
> -  ~~StringProgressBar: https://pypi.org/project/StringProgressBar~~</br>
> -  All Pip Requirements are in requirements.txt
> -  MakeMKV (MakeMKV is free while in beta however keys have to be obtained every 60 days: https://forum.makemkv.com/forum/viewtopic.php?t=1053)

### Installation
#### Running from Source
-  Download the repository using ```git clone https://github.com/ItBePhill/MakeMKV-Auto.git``` or by manually downloading
-  run ```pip install -r requirements.txt```
-  run ```python Main.py```
-  close the program
-  edit the config.ini file to your needs
-  rerun ```python Main.py```
-  and enjoy
