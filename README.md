# Make MKV-Auto
---------------------------------------------------------------
<p>Automatically finds and then rips blurays and dvds based on options in an ini file</p>




> [!NOTE]
> Instructions: <br>
> run first to generate a config.ini file then edit the file and relaunch<br>
>
> Credits:<br>
> open-tray.py is edited from: https://gist.github.com/TakesTheBiscuit/a2e3d36c1d20731821fdb41b3831406e#file-open_and_close_cd_tray-py

> [!WARNING]
> Only Works on Windows However I will gladly review a pr to fix that

> [!WARNING]
> Required Modules:<br>
> StringProgressBar: https://pypi.org/project/StringProgressBar/<br>

> Also requires MakeMKV<br>
> MakeMKV is free while in beta however keys have to be obtained every 60 days: https://forum.makemkv.com/forum/viewtopic.php?t=1053
